# 🎬 MowiTajm

A web application where users can search for movies and write reviews.

## Features

- **Movie Search** – Search for movies via the OMDB API and view detailed information
- **Reviews** – Logged-in users can write, edit, and delete their own reviews
- **Ratings** – Rate movies 1–5 stars
- **User Management** – Registration, login, and role-based access (Admin / User)
- **Admin Panel** – Administrators can manage users and roles
